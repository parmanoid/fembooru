# 🎨 Booru Board

Аниме-борда в стиле Danbooru/Gelbooru, работающая на GitHub Pages + Firebase.

## 📁 Структура файлов

```
/
├── index.html          — Главная лента
├── post.html           — Страница поста
├── profile.html        — Профиль пользователя
├── upload.html         — Загрузка поста
├── login.html          — Вход / Регистрация
├── 404.html            — Страница 404
├── .nojekyll           — Отключает Jekyll
├── css/
│   └── style.css       — Все стили
├── js/
│   ├── app.js          — Firebase + общие утилиты
│   ├── feed.js         — Логика ленты
│   ├── post-page.js    — Страница поста
│   ├── profile-page.js — Профиль
│   ├── upload-page.js  — Загрузка
│   └── login-page.js   — Авторизация
└── database.rules.json — Правила Firebase
```

## 🚀 Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы в корень репозитория
3. Перейдите: **Settings → Pages → Source: Deploy from a branch → main / root**
4. Сайт будет доступен по адресу `https://your-username.github.io/repo-name/`

## 🔥 Настройка Firebase

### Realtime Database Rules

1. Откройте [Firebase Console](https://console.firebase.google.com/)
2. Выберите ваш проект → **Realtime Database → Rules**
3. Вставьте содержимое файла `database.rules.json`
4. Нажмите **Publish**

### Auth настройки (уже сделано по условию)

- Authentication → Sign-in method → Email/Password: **включено**

### Database indexes (рекомендуется)

В консоли Firebase → Realtime Database → Rules добавьте индексы (уже включены в rules).

## 📝 Использование

### Логин
- Введите любой логин (3-20 символов, a-z, 0-9, _)
- Под капотом будет `логин@anon.ru`

### Загрузка поста
1. Выберите изображение или видео
2. Добавьте теги (Enter или запятая для разделения)
3. Напишите описание с опциональными #тегами и @упоминаниями
4. Нажмите «Опубликовать»

### Поиск
- Введите тег в строку поиска и нажмите Enter
- Можно добавить несколько тегов для уточнения поиска
- Выпадающий список показывает теги с количеством постов
